# When to use
Manually

# Description
You must install this project according to the instructions in README.md file, the Installation section. Do the following yourself:
- Copy the `.env.example` file and rename it to `.env`, don't change anything in `.env`.
- Run the command `make init`.

If there are errors during installation, display them to the user and tell them how to solve them. Ask for permission to solve bugs yourself and rebuild the project.

If the installation was successful, display the message "The project was successfully launched, if you want to install the Laravel Starter Kit, run skill laravel-starter-kit"
